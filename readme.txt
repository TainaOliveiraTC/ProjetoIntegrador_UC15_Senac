|| Titulo do projeto: CaféOliver
|| Status do projeto: Em desenvolvimento.
|| Tecnologias que serão aplicadas: Linguagem - Java | Banco de dados - MYSQL
|| Time de desenvolvedores: Tainá Oliveira
|| Objetivo do software: Criar um sistema desktop offline que auxilie o dono do café a organizar suas vendas, seu financeiro e os seus funcionários.
|| Funcionalidades do sistema: 
    *Registro de ingredientes
    *Registro de produtos
    *Registro de vendas
    *Registro de produtos já feitos
    *cadastro de funcionários
    *Registro de despesas
    *Tela de listagem para mostrar pedidos
    *Tela de Menu
    *Tela do financeiro
    *Tela de cadastro de login
    *Tela de login 
    *Tela Principal
    *Banco de dados próprio.
